## 🚀 Featured Project: AI Playground (A GenAI Hub)

**🔗 Live Demo:** [https://aiplayground.mohitpatel.org](https://aiplayground.mohitpatel.org)
**📦 Source:** [github.com/moohiit/ai-playground](https://github.com/moohiit/ai-playground)

A unified platform hosting **six end-to-end generative AI projects** — built to demonstrate practical GenAI engineering: retrieval-augmented generation, embeddings, streaming, structured output, vision, prompt design, and clean APIs.

**🧠 Projects inside:**
- **Resume ↔ JD Matcher** — structured-output scoring with matched/missing skills and tailored bullet rewrites
- **Natural Language → SQL Generator** — prompt + regex-guarded `SELECT`-only queries executed in an in-memory SQLite sandbox
- **Expense Tracker** — receipt scanning via Gemini Vision, group splitting, settlement math, Recharts dashboards
- **AI Content Generator** — streamed Markdown articles, editable outlines, SEO meta + Twitter thread + LinkedIn post derivatives
- **PDF Chat (RAG)** — upload a PDF, ask grounded questions with page-level `[#n]` citations
- **YouTube Q&A (RAG)** — paste a URL, ask grounded questions with timestamp citations that seek the embedded player

**🔐 Platform features:**
- JWT auth with email verification (Resend) and password reset via OTP
- Profile page with Cloudinary photo upload and per-user monthly AI request limits
- Per-project rate limits and usage logging in MongoDB

**🛠️ Built With:**
- **Frontend:** Next.js 14 (App Router), Tailwind, react-markdown, YouTube IFrame API
- **Backend:** Next.js Route Handlers, Node.js
- **AI:** Google Gemini (2.5 Flash, Flash-Lite, Vision, Embedding-001), structured output, streaming
- **Databases:** MongoDB Atlas + **Vector Search** for RAG, in-memory SQLite sandbox
- **Infra:** Cloudinary (photos), Resend (transactional email), Vercel
- **Auth & Safety:** JWT, bcrypt, Zod validation, per-user + per-IP rate limits
